Dreamer's Canal

Group Project (of 3) - Programming, Art, UI & Tech Design, Production
Tools Used: Unity, Aseprite, Trello/Discord
Project Goals: Build a functioning game for the GMTK Game Jam (theme: Loop), improve prototype iterations quickly
Time span: 4 days




Dreamer's Canal is a roguelike trading game about a merchant selling wares along a magical looping canal.
This project expanded my prototyping skills, communication skills due to the number of testers involved and my scope management given the short time frame.
