PENGUN

Solo Project - Design, Programming, Art, SFX
Tools Used: Unity, Aseprite, JSFXR
Project Goals: Learn Unity, expand knowledge of C#, improve pixel art skills.
Time span: 1 month

Pengun is a top-down 2D arena shooter inspired by Enter The Gungeon and Downwell.
This project was created in my C# For Games Designers module. I handled the design, mechanics, art, sound effects, and programming.
Music by Ben Reber.
